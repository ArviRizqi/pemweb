# 🌐 Praktik Pemrograman Web (JavaScript, PHP, CSS)

## 📌 Deskripsi Proyek
Proyek ini merupakan implementasi dasar hingga menengah dalam **pemrograman web** menggunakan teknologi inti:
- **JavaScript** (interaktivitas)
- **PHP** (backend/server-side)
- **CSS** (tampilan)

Aplikasi ini dirancang dengan pendekatan modular, di mana setiap bagian (frontend dan backend) bekerja secara terpisah namun saling terintegrasi melalui HTTP request.

---

## 🎯 Tujuan
- Memahami dasar pengembangan web fullstack
- Mengintegrasikan frontend dan backend
- Mengelola data menggunakan PHP
- Membangun UI yang responsif dengan CSS
- Menggunakan JavaScript untuk interaksi dinamis

---

## 🚀 Fitur Utama
- 🔐 Sistem autentikasi (login & register)
- 📄 CRUD data (Create, Read, Update, Delete)
- 🔄 Interaksi dinamis menggunakan JavaScript (AJAX / Fetch API)
- 🎨 Tampilan responsif dengan CSS
- 🗄️ Koneksi database (MySQL)

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla JS)

### Backend
- PHP (Native / tanpa framework)

### Database
- MySQL
